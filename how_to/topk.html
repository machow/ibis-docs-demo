Compute the top K records

Here we use the [topk][ibis.expr.types.Column.topk] method to compute the top 5 customers for some generated TPC-H data by:

  • count (the default)
  • sum of order totals
>>> import ibis
>>> ibis.options.interactive = True
>>> con = ibis.duckdb.connect()  # in-memory duckdb
>>> con.raw_sql("CALL dbgen(sf=0.1)")
>>> orders = con.table("orders")
>>> orders.o_custkey.topk(5)  # top 5 most frequent customers
