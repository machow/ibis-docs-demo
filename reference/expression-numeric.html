expr.types.numeric.NumericColumn.bucket(self, buckets, closed='left', close_extreme=True, include_under=False, include_over=False)

Parameters

Name Type Description Default
buckets typing.Sequence[int] List of buckets required
closed typing.Literal[‘left’, ‘right’] Which side of each interval is closed. For example: python buckets = [0, 100, 200] closed = "left" # 100 falls in 2nd bucket closed = "right" # 100 falls in 1st bucket 'left'
close_extreme bool Whether the extreme values fall in the last bucket True
include_over bool Include values greater than the last bucket in the last bucket False
include_under bool Include values less than the first bucket in the first bucket False

Returns

Type Description
IntegerColumn A categorical column expression
