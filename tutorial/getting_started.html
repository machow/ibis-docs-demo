Getting started with ibis

This is a quick tour of some basic commands and usage patterns, just to get your flippers wet.

Install ibis

This quick-start guide uses the DuckDB backend. You can check out the Install page for information on how to install other backends.

shell title="Install Ibis using pip" $ pip install 'ibis-framework[duckdb]'

shell title="Install Ibis using conda" $ conda install ibis-framework

Download a database file

Ibis can work with several file types, but at its core, it connects to existing databases and interacts with the data there. Weâ€™ll use a local database (DuckDB) to get the hang of this.1
